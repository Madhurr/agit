Command: agit init

No flags needed.

Behavior:
1. Get working dir with os.Getwd()
2. Check git.IsRepo(dir) — if false, print error "not a git repository" and return error
3. Run these git config commands via git.RunGit:
   a. git config --add remote.origin.fetch "+refs/notes/agit:refs/notes/agit"
      (so git fetch also fetches agit notes)
      Ignore error on this one if origin doesn't exist yet.
   b. git config notes.rewriteRef "refs/notes/agit"
      (so git rebase/cherry-pick carries notes forward)
4. Print success output:
   green "✓ agit initialized in this repository"
   normal ""
   normal "To push notes to remote:"
   cyan   "  git push origin refs/notes/agit"
   normal ""
   normal "To fetch notes from remote:"
   cyan   "  git fetch origin refs/notes/agit:refs/notes/agit"
   normal ""
   normal "Agents should now use 'agit commit' instead of 'git commit'."
